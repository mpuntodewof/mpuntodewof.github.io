

## Personal Landing Page — Full-Stack Engineer

A premium, dark-themed SaaS-style landing page with blue/purple gradients, glassmorphism, and subtle animations. The uploaded profile image will be embedded in the hero/about section.

### Sections (10 total)

1. **Hero** — Headline "Full-Stack Engineer Building Scalable Backend Systems", dual CTAs (Hire Me / View Resume), profile image, animated background with architecture/data flow visual
2. **Trust & Impact** — Animated counter metrics (1000+ transactions, 40% error reduction, enterprise systems delivered)
3. **Services** — 4 cards (Backend/API, Full-Stack, System Integration, Performance) with problem→solution→outcome format, "Start a Project" CTA
4. **Experience** — Timeline of professional roles (Banking, Enterprise, Full-stack), "Download Resume" CTA
5. **Featured Projects** — Talenta API, StockLedger, Habit Tracker, Banking APIs with tech stack tags and impact metrics
6. **Tech Stack** — Grouped grid (Backend, Frontend, Database, Tools) with icons/logos
7. **Work Process** — 4-step visual flow (Understand → Design → Build → Optimize)
8. **About** — Profile image, personal positioning statement, backend-heavy full-stack narrative
9. **Final CTA** — Split section: Left for clients ("Start a Project"), Right for recruiters ("View Resume")
10. **Footer** — GitHub, LinkedIn, Email links

### Design System
- **Theme**: Dark background (#0a0a1a → #141432), blue/purple gradient accents (#4f46e5, #7c3aed, #3b82f6)
- **Typography**: Space Grotesk headings, DM Sans body
- **Effects**: Glassmorphism cards, subtle fade/slide-in animations, gradient borders
- **Style**: Stripe/Linear/Vercel-inspired premium SaaS feel

### Assets
- Profile image embedded from upload
- CSS-based architecture diagrams and data flow visuals (no external images needed)
